Key Features & Contributions

Custom ACF Blocks for State Cards

I created a custom post type (CPT) named "State Cards" and implemented a query to loop through all posts, displaying information such as state name, image, city names within the state, and a designated color picker. This CPT was used across the entire site for various components, including the display of service area cards. When users toggled the plus icon the card opened revealing the city names tagged with its page link. This was made possible using CSS and Javascript.

Process

  • Created CPT using “CPT UI” plugin
  • Created new ACF group that will house the States information
  • Built query to loop through correct post type
  • Built service cards using html, php, and styled using css
  • Added JS for on-click toggl capability

Custom ACF Blocks for Heros

I developed a custom block using PHP and ACF fields, enabling the client to easily add images and titles through the data field. This block dynamically displays the background image with an overlaid text on the front end. On mobile devices, the content repositions above the image for an enhanced responsive user experience.

Process

  • Created new ACF group adding in necessary fields for the block
  • Built custom block using html and styled using css
  • Made block dynamic by adding in the acf fields
  • Revised Mobile View
