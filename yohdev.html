Key Features & Contributions

Mailchimp Gracity Forms Intergration

Designed and implemented user-friendly forms using WordPress’ Gravity Forms plugin. Additionally, I set up and managed a Mailchimp account, incorporating tags to efficiently route email signups to the appropriate campaigns.

Process

  • Installed and configured the Gravity Forms plugin
  • Designed and styled a new form
  • Embedded the form on the designated page/component using Gravity Forms shortcode
  • Created and configured tags in Mailchimp
  • Linked form submissions to the appropriate Mailchimp tags for effective campaign management

Custom Category Tabs Archive for Technologies Used

Developed custom HTML block that dynamically displays technology card components pulled from the Technologies CPT(custom post type) based on user interaction with the tab navigation.

Process

  • Created a custom post type (CPT) named "Technologies"
  • Developed a custom HTML block and styled it with CSS
  • Implemented a query loop to iterate through the "posts"/technologies
  • Retrieved image, title, and excerpt

Custom Vertical Tabs Block

Utilized Bootstrap’s vertical tabs layout and customized it to fit the company's needs. YohDev was looking for a slick, user interactive component that would showcase their services.

Process

  • Customized Bootstrap’s tabbed vertical navigation block
  • Styled the block to match the company’s theme
  • Revised mobile view
  • Converted the block into a reusable component
