Key Features & Contributions

Page Builds

Using the designed Figma file, I replicated each page and transformed it into an interactive site. While Elementor Pro offers many templates, this site was custom-built using the drag-and-drop builder along with extensive CSS.

Process

  • Built each page using the drag-and-drop editor
  • Styled using Elementor style settings and CSS
  • Revised Mobile View

Repeated Components

Many of the featured components are utilized throughout the site. My task was to efficiently create and save templates for use on different pages, making it easier and faster to build and maintain the site.

Process

  • Built and styled in Elementor
  • Saved as a reusable component using the saved templates feature
