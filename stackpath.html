Key Features & Contributions

Header/Footer Build

I built a custom header navigation that allows for the dynamic use of WordPress menus, making it easy for the client to update menu items as needed. This multi-level mega-menu serves as an eye-catching, interactive navigation feature. The footer also supports WordPress menus. Both components can collapse multi-menu items in mobile view, ensuring a responsive site.

Process

  • Built static navigations and ensured proper styling passed QA
  • Created header and footer menus in WordPress
  • Registered menus in the functions.php file
  • Used wp_nav_menu to plug in the menu items, enabling dynamic functionality
